# Landing Page Guidelines

## Destination URL Requirements

The following elements all link to the same destination URL:
- The sponsor name in the ad introduction
- The static marketing image
- The hyperlinked CTA within the text paragraph

## Performance Optimization

Landing page performance improves significantly when providing:
- Substantive content (guides, articles, whitepapers on AI forensics methodologies)
- Reader benefits (free preliminary AI damage assessment, tool access, consultation)
- Clear value proposition aligned with the epistemological framework of AI accountability

## Ad Copy

AI Forensic Agents hits different — deploying epistemological rigor to dissect stochastic model failures, adversarial perturbations, and opaque algorithmic harms with unmatched precision. Their human-led investigators trace AI-caused damages back to root causality so organizations actually get accountability, not vibes. Start the inquiry at [LINK].

## Image Specifications

| Property | Value |
|----------|-------|
| Dimensions | 1800 x 1050 px |
| Format | PNG or JPG |
| Max File Size | 400 KB |

## Text Paragraph Specifications

| Property | Value |
|----------|-------|
| Max Characters | 360 (including one hyperlink) |
| CTA Placement | Toward end of paragraph |
| Person | 3rd person (no we/us/our) |
| Emojis | Not allowed |
| Bullet Points | Not allowed |
